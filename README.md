# 💰 FinançasClaras

Projeto desenvolvido para o **Itinerário Extensionista**, com o objetivo de conscientizar e educar pessoas sobre a importância de investir e cuidar da vida financeira.

---

## 📌 Sobre o Projeto

O **FinançasClaras** é um site informativo focado em educação financeira, trazendo conteúdos simples e diretos sobre:

- Organização financeira
- Saída de dívidas
- Construção de reserva de emergência
- Introdução aos investimentos

A proposta é tornar o conhecimento financeiro acessível para qualquer pessoa, mesmo sem experiência.

---

## 🎯 Objetivo

Ajudar usuários a:

- Entender a importância de investir
- Criar hábitos financeiros saudáveis
- Desenvolver segurança financeira ao longo do tempo

---

## 🧮 Calculadoras Disponíveis

O site conta com ferramentas práticas para auxiliar no planejamento financeiro:

- 💸 **Calculadora de Reserva de Emergência**  
  Ajuda a descobrir quanto guardar com base no custo de vida mensal.

- 📈 **Calculadora de Juros Compostos**  
  Simula o crescimento do dinheiro ao longo do tempo com investimentos.

- 🏆 **Calculadora do Primeiro Milhão**  
  Mostra quanto tempo leva para atingir R$1.000.000 com aportes mensais.
